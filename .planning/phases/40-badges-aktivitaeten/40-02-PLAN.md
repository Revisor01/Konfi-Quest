        3. Alle Aktivitaets/Event-Daten sammeln
        4. Fuer jedes Jahr seit Transition: Zaehle ob mind. 1 Eintrag vorhanden

1. In teamer.js neue Endpunkte hinzufuegen:

   GET /api/teamer/badges -- Alle verfuegbaren Teamer-Badges mit earned-Status:
   ```sql
   SELECT cb.*,
     CASE WHEN ub.id IS NOT NULL THEN true ELSE false END as earned,
     ub.awarded_date
   FROM custom_badges cb
   LEFT JOIN user_badges ub ON cb.id = ub.badge_id AND ub.user_id = $1
   WHERE cb.organization_id = $2 AND cb.target_role = 'teamer' AND (cb.is_active = true OR ub.id IS NOT NULL)
   ORDER BY ub.awarded_date DESC NULLS LAST, cb.name
   ```

   GET /api/teamer/badges/unseen -- Anzahl ungesehener Badges:
   ```sql
   SELECT COUNT(*) FROM user_badges WHERE user_id = $1 AND organization_id = $2 AND seen = false
   ```

   PUT /api/teamer/badges/mark-seen -- Badges als gesehen markieren:
   ```sql
   UPDATE user_badges SET seen = true WHERE user_id = $1 AND organization_id = $2 AND seen = false
   ```

2. In activities.js POST /assign-activity:
