<interfaces>
<!-- Migrations-API: Drop-In-Ersatz fuer useHistory -->

Migration Pattern (IMMER so umsetzen):
```typescript
// ENTFERNEN:
import { useHistory } from 'react-router-dom';
const history = useHistory();

// HINZUFUEGEN:
import { useIonRouter } from '@ionic/react';
const router = useIonRouter();
```

API-Aequivalenz:
- history.push('/pfad')            → router.push('/pfad')
- history.replace('/pfad')         → router.push('/pfad', 'root', 'replace')
- history.goBack()                 → router.goBack()
- canGoBack() check                → router.canGoBack()

State uebergeben:
- history.push('/pfad', state)     → router.push('/pfad', 'forward', 'push', undefined, state)

Kommentar in jede migrierte Datei einfuegen (direkt nach dem Import):
// useIonRouter: Ionic 8 API - bei Ionic v9 ggf. auf useNavigate migrieren
</interfaces>
