    Export: `openFileNatively(blobOrUrl: Blob | string, fileName: string, mimeType: string): Promise&lt;boolean&gt;`

    Logik:
    1. `if (!Capacitor.isNativePlatform()) return false;` — Web-Fallback sofort
    2. Blob beschaffen: Wenn `blobOrUrl` ein string ist (URL), per `fetch(blobOrUrl)` laden. Wenn bereits Blob, direkt verwenden.
    3. FileReader → base64 (gleicher Code wie MaterialFormModal Zeile 170-177)
    4. `Filesystem.mkdir({ path: 'temp', directory: Directory.Documents, recursive: true })` (catch ignorieren falls existiert)
    5. `const ext = fileName.split('.').pop() || 'bin';`
    6. `const tempPath = 'temp/native_' + Date.now() + '.' + ext;`
    7. `Filesystem.writeFile({ path: tempPath, data: base64, directory: Directory.Documents, recursive: true })`
    8. `const fileUri = await Filesystem.getUri({ directory: Directory.Documents, path: tempPath });`
    9. `if (mimeType.startsWith('image/'))` → `FileOpener.open({ filePath: fileUri.uri, contentType: mimeType })` (per D-02)
    10. `else` → `FileViewer.openDocumentFromLocalPath({ path: fileUri.uri })` (per D-03)
    11. `return true;`
    12. `catch` → `console.warn('Natives Oeffnen fehlgeschlagen:', err); return false;` (per D-04: Caller nutzt FileViewerModal als Fallback)

    Imports: Capacitor aus @capacitor/core, Filesystem/Directory aus @capacitor/filesystem, FileOpener aus @capacitor-community/file-opener, FileViewer aus @capacitor/file-viewer.
