1. In backend/server.js einen neuen Rate-Limiter nach den bestehenden Limitern (nach Zeile 199) definieren:
   ```javascript

   const orgLimiter = rateLimit({
     windowMs: 15 * 60 * 1000, // 15 Minuten

   ```

2. Den orgLimiter als Parameter an die organizations Route uebergeben. Die Route-Registrierung in server.js (ca. Zeile 419) anpassen:

HINWEIS: 20 Requests pro 15 Minuten ist ausreichend -- Org-Verwaltung ist ein seltener Admin-Vorgang. GET-Requests zaehlen mit, da auch Statistik-Abrufe limitiert werden sollen.
