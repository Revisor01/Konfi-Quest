<!-- Referenz-Pattern aus Konfi ActivityRequestModal (Phase 60): -->
```typescript
// Online/Offline-Branching im Handler:
if (networkMonitor.isOnline) {
  // Online-Pfad: direkt senden (bestehender Code)
  try { await api.post(...); onSuccess(); } catch { setError(...); }
} else {
  // Offline-Pfad: Queue-Fallback
  await writeQueue.enqueue({
    method: 'POST',
    url: '/konfi/requests',
    body: queueBody,
    maxRetries: 5,
    hasFileUpload: false,
    metadata: { type: 'request', clientId, label: 'Aktivitaets-Antrag' },
  });
  setSuccess('...wird gesendet sobald du wieder online bist');
  onSuccess();
}
```
