Fuer jede der 17 Dateien: Die bestehende einzelne `app-corner-badge` div in einen neuen `app-corner-badges` Container wrappen.

**Muster — Vorher (einzelner Badge):**
```tsx
<div className="app-corner-badge app-corner-badge--events">Text</div>
// oder:
<div className="app-corner-badge" style={{ backgroundColor: color }}>Text</div>
```

**Muster — Nachher:**
```tsx
<div className="app-corner-badges">
  <div className="app-corner-badge app-corner-badge--events">Text</div>
</div>
// oder:
<div className="app-corner-badges">
  <div className="app-corner-badge" style={{ backgroundColor: color }}>Text</div>
</div>
```

**WICHTIG bei der Migration:**
- `position: absolute; top: 0; right: 0;` wird NICHT mehr am Badge selbst gebraucht — das kommt vom Container. Falls ein Badge diese inline-styles hat, ENTFERNEN.
- `position: static` (das manche Badges haben fuer den alten Flex-Container) ist ebenfalls nicht mehr noetig — ENTFERNEN.
- `border-radius` inline-styles am Badge ENTFERNEN — die Rundung kommt jetzt per CSS :last-child/:only-child.
- `style={{ backgroundColor: color }}` BEIBEHALTEN (dynamische Farben).
