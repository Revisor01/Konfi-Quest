## Interaction Contract

### Queue-Badge Lebenszyklus

| Zustand | Darstellung | Uebergang |
|---------|-------------|-----------|
| Pending (in Queue) | Uhr-Icon, orange #ff9500 | Erscheint sofort bei Queue-Einreihung |
| Zugestellt (Erfolg) | Badge verschwindet | Kein Haekchen, kein Feedback (D-11) |
| Permanenter Fehler (4xx) | Ausrufezeichen, rot #dc3545 | Icon-Wechsel + Farbwechsel (D-12) |

### Fehler-Badge Tap-Interaktion (OUI-07)

Bei Tap auf Fehler-Badge oeffnet sich ein `IonActionSheet` mit:

| Button | Rolle | Aktion |
|--------|-------|--------|
| "Erneut senden" | — | Queue-Item wird erneut eingereiht |
| "Loeschen" | destructive | Queue-Item wird entfernt, optimistische UI wird zurueckgerollt |
| "Abbrechen" | cancel | ActionSheet schliesst sich |

### Badge-Reihenfolge im Container (links nach rechts)

1. Queue-/Fehler-Badge (optional, nur wenn Item in Queue oder fehlgeschlagen)
2. Trenner (2px weiss, nur wenn Queue-Badge UND weitere Badges vorhanden)
3. Bestehende Status-Badges (Typ, Kategorie, Punkte, etc.)

### Title-Padding bei Badges

| Anzahl Badges | paddingRight auf `__title` |
|---------------|---------------------------|
| 0 | 0 |
| 1 | 70px (bestehend) |
| 2 | 120px (bestehend) |
| 3 | 160px |

## Migration: Betroffene Dateien

Alle ~21 Dateien mit bestehenden Corner-Badges muessen migriert werden. Die Migration ist rein strukturell:

**Vorher (einzelner Badge):**
```tsx
<div className="app-corner-badge app-corner-badge--events">
  Text
</div>
```

**Nachher (im Container):**
```tsx
<div className="app-corner-badges">
  <div className="app-corner-badge app-corner-badge--events">
    Text
  </div>
</div>
```

**Vorher (PointsHistory dual inline-styles):**
```tsx
<div style={{ position: 'absolute', top: '0', right: '0', display: 'flex', ... }}>
  <div style={{ backgroundColor: color, ... }}>Text1</div>
  <div style={{ width: '2px', background: 'white' }} />
  <div className="app-corner-badge" style={{ position: 'static', ... }}>Text2</div>
</div>
```

**Nachher (CSS-Klassen):**
```tsx
<div className="app-corner-badges">
  <div className="app-corner-badge" style={{ backgroundColor: color }}>
    Text1
  </div>
  <div className="app-corner-badges__separator" />
```
