   - `shareSlide(cardElement: HTMLElement, slideKey: string, wrappedType: 'konfi' | 'teamer', textFallbackData: ShareTextData): Promise<void>`
   - **Bild-Export:** `toPng(cardElement, { width: 1080, height: 1920, pixelRatio: 1, cacheBust: true })` aus html-to-image
   - **Native Share (Capacitor):** Wenn `Capacitor.isNativePlatform()`:
     a. Base64 aus dataUrl extrahieren (dataUrl.split(',')[1])
     b. `Filesystem.writeFile({ path: 'wrapped_share.png', data: base64, directory: Directory.Cache })`
     c. `Filesystem.getUri({ path: 'wrapped_share.png', directory: Directory.Cache })`
     d. `Share.share({ files: [uri.uri] })`
   - **Web-Fallback:** Wenn NICHT nativ: Download-Link erstellen (a-Element mit download-Attribut + dataUrl als href, click(), remove())
   - **Text-Fallback:** Im catch-Block: `Share.share({ title: 'Mein Konfi Wrapped', text: generateFallbackText(textFallbackData) })` — auf Web: `navigator.share({ text })` oder `navigator.clipboard.writeText(text)`
   - Interface `ShareTextData`: `{ wrappedType, displayName, year, slideKey, slideValue?: string }` — slideValue ist der Hauptwert des Slides (z.B. "847 Punkte")
   - `generateFallbackText()` erzeugt Satz wie: "Mein Konfi-Jahr 2026: 847 Punkte gesammelt! #KonfiQuest"
   - Imports: `import { toPng } from 'html-to-image'`, `import { Capacitor } from '@capacitor/core'`, `import { Filesystem, Directory } from '@capacitor/filesystem'`, `import { Share } from '@capacitor/share'`
