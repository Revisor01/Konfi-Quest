In `backend/push/firebase.js`, Zeilen 65-67 (catch-Block von sendFirebasePushNotification) aendern:

VORHER:
```javascript
} catch (error) {
    console.error('Firebase notification error:', error);
    return { success: false, error: error.message };
}
```

NACHHER:
```javascript
} catch (error) {
    console.error('Firebase notification error:', error);
    return { success: false, error: error.message, errorCode: error.code || null };
}
```

Einzige Aenderung: `errorCode: error.code || null` zum Return-Objekt hinzufuegen. Null-safe da nicht alle Errors einen .code haben (z.B. Netzwerk-Fehler). Phase 26 wird diesen errorCode auswerten um Tokens zu loeschen oder error_count zu erhoehen.
