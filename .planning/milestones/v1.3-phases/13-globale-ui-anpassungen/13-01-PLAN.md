  In frontend/src/theme/variables.css folgende Aenderungen vornehmen:

  **GUI-01: Listen-Icons kleiner und oben positioniert**
  1. `.app-icon-circle--lg` (ca. Zeile 364): width und height von 40px auf 32px aendern
  2. `.app-icon-circle--lg ion-icon` (ca. Zeile 388): font-size von 1.2rem auf 1rem aendern
  3. `.app-list-item__main` (ca. Zeile 438): align-items von `center` auf `flex-start` aendern -- damit Icons am oberen Rand des Listeneintrags positioniert sind, nicht mittig
  4. Neuen Modifier `.app-list-item__main--centered` hinzufuegen mit `align-items: center` fuer Faelle wo zentriert gewuenscht ist (Fallback)

  **GUI-02: Auswahl-Rahmen bei Fokus/Select entfernt**
  Neue globale CSS-Regeln NACH den bestehenden Popover-Styles (ca. Zeile 558) einfuegen:
  ```css
  /* === Fokus/Auswahl-Rahmen global entfernt (GUI-02) === */
  ion-checkbox {
    --checkbox-background-checked: var(--ion-color-primary, #3880ff);
    --border-color-checked: var(--ion-color-primary, #3880ff);
  }
  ion-checkbox::part(container) {
    border-radius: 4px;
  }
  /* Fokus-Outline bei Items und Checkboxen entfernen */
  ion-item::part(native):focus,
  ion-item::part(native):focus-visible {
    outline: none;
    box-shadow: none;
  }
  ion-checkbox:focus,
  ion-checkbox:focus-visible,
  ion-radio:focus,
  ion-radio:focus-visible {
    outline: none;
    box-shadow: none;
  }
  /* iOS/Android Highlight-Farbe bei focused Items neutralisieren */
  ion-item {
    --highlight-color-focused: transparent;
    --highlight-color-valid: transparent;
    --highlight-color-invalid: transparent;
  }
  ```

  **GUI-05: Auth-Seiten durchgehende Hintergrundfarbe**
  Die `.app-auth-background` Klasse (ca. Zeile 1164) erweitern:
  ```css
  .app-auth-background {
    --background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
    --overflow: auto;
  }
  .app-auth-background::part(scroll) {
    min-height: 100%;
  }
  ```
  Ausserdem `.app-auth-container` anpassen:
  ```css
  .app-auth-container {
    display: flex;
    flex-direction: column;
    justify-content: center;
    align-items: center;
    min-height: 100%;
    padding: 20px;
    padding-bottom: env(safe-area-inset-bottom, 20px);
  }
  ```
  Der Wechsel von `min-height: 100vh` zu `min-height: 100%` zusammen mit dem `::part(scroll)` Styling stellt sicher, dass der Gradient den gesamten sichtbaren Bereich abdeckt und kein weisser Bereich am unteren Rand erscheint. Das `padding-bottom: env(safe-area-inset-bottom)` sorgt fuer korrektes Spacing auf Geraeten mit Home-Indikator.
