## Goal

Level-Modal verwendet Lila Icon-Farbe und hat iOS Backdrop-Effekt. Profil-Modals zeigen Icons und Funktionsbeschreibung in Lila.

## must_haves

- [ ] LevelManagementModal section-icon ist Lila (app-section-icon--purple oder --level) statt Settings-Blau (SET-06)
- [ ] LevelManagementModal submit-Button verwendet `--level` statt `--settings` CSS-Klasse (SET-06)
- [ ] AdminProfilePage Profil-Items Icons und -Farbe bleiben Lila (SET-07 -- Verifikation)

## Context

**LevelManagementModal.tsx** (`frontend/src/components/admin/modals/LevelManagementModal.tsx`):
- Aktuell: `app-section-icon--settings` (Zeile 280) = #667eea (Blau)
- Soll: `app-section-icon--level` = #5b21b6 (Lila)
- Submit-Button: `app-modal-submit-btn--settings` (Zeile 264) -- soll zu `--level` werden fuer Lila-Farbe
- iOS Backdrop-Effekt: Der Modal wird per `useIonModal` praesentiert mit `presentingElement` -- das erzeugt bereits den iOS card-style Backdrop. Pruefen ob das korrekt funktioniert.

**AdminProfilePage.tsx** (`frontend/src/components/admin/pages/AdminProfilePage.tsx`):
- Aktuell verwendet bereits `app-section-icon--purple` und `app-icon-circle--purple` und `app-list-item--purple` -- ist bereits Lila
- SET-07 scheint bereits implementiert -- VERIFIZIEREN und ggf. kleinere Korrekturen

**CSS submit-Button Klassen** (`frontend/src/theme/variables.css`):
- `app-modal-submit-btn--settings` hat Farbe #667eea (Blau)
- Braucht neue Klasse `app-modal-submit-btn--level` mit Farbe #5b21b6 (Lila), ODER den bestehenden `--purple` Modifier nutzen falls vorhanden

## Tasks

<task id="18-02-1">
<title>SET-06: LevelManagementModal auf Lila + Backdrop</title>
<instructions>
In `frontend/src/components/admin/modals/LevelManagementModal.tsx`:

1. **Section-Icon von Settings auf Level aendern** (Zeile 280):

Aktuell:
```tsx
<div className="app-section-icon app-section-icon--settings">
```

Aendern zu:
```tsx
<div className="app-section-icon app-section-icon--level">
```

Dies aendert die Farbe von #667eea (Blau) zu #5b21b6 (Lila).

2. **Submit-Button Farbe anpassen** (Zeile 264):

Aktuell:
```tsx
className="app-modal-submit-btn app-modal-submit-btn--settings"
```

Aendern zu:
```tsx
className="app-modal-submit-btn app-modal-submit-btn--level"
```

Pruefe in `frontend/src/theme/variables.css` ob `app-modal-submit-btn--level` existiert. Falls nicht, fuege hinzu:

```css
.app-modal-submit-btn--level {
  --color: #5b21b6;
}
```

Fuege dies NACH dem bestehenden `app-modal-submit-btn--settings` Block ein.

3. **iOS Backdrop-Effekt verifizieren:** Der Modal wird ueber `useIonModal` mit `presentingElement` geoeffnet (in `AdminLevelsPage.tsx` Zeile 152: `presentLevelModal({ presentingElement })`). Das erzeugt automatisch den iOS card-style presentation. Keine Aenderung noetig, nur VERIFIZIEREN.

**Ergebnis:** Level-Modal hat Lila Icons und Submit-Button, iOS Backdrop funktioniert.
</instructions>
</task>
