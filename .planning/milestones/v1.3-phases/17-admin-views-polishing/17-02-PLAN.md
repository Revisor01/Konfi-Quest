## Goal

Benutzer:innen-Liste verwendet einheitlich Solid-Icons und GoalsPage nutzt das Design-System-Stepper-Pattern.

## must_haves

- [ ] UsersView: briefcase (Solid) statt briefcaseOutline fuer role_title-Anzeige (AUI-03)
- [ ] GoalsPage: Standard-Stepper-Pattern (IonButton fill="outline" mit IonInput) statt app-stepper CSS-Klassen (AUI-04)

## Context

**UsersView.tsx** (`frontend/src/components/admin/UsersView.tsx`):
- Zeile 30: `briefcaseOutline` ist importiert -- soll durch `briefcase` (Solid) ersetzt werden
- Zeile 281: `<IonIcon icon={briefcaseOutline}` -- zeigt die Funktionsbeschreibung (role_title) als Line-Icon
- Das Requirement will Solid-Icons statt Line-Icons fuer die Funktionsanzeige
- Auch `peopleOutline` (Zeile 32) wird fuer den Listen-Header verwendet -- das ist OK (Section-Icons sollen Outline sein laut Design-System)

**AdminGoalsPage.tsx** (`frontend/src/components/admin/pages/AdminGoalsPage.tsx`):
- Verwendet `app-stepper`, `app-stepper__value`, `app-stepper__button` CSS-Klassen (Zeilen 119-170)
- Das Design-System definiert einen Standard-Stepper mit IonButton fill="outline" + IonInput (siehe DESIGN_SYSTEM.md Zahlen-Stepper)
- Aktuell: Eigene CSS-Klassen mit `IonButton fill="clear"` und `div.app-stepper__value`
- Soll auf: `IonButton fill="outline"` + `IonInput type="text" inputMode="numeric"` umgestellt werden

2. **Gottesdienst-Stepper ersetzen** (ca. Zeile 118-142). Ersetze den gesamten Block innerhalb `<IonCardContent>`:

Aktuell:
```tsx
<IonCardContent>
  <div className="app-stepper">
    <IonButton
      fill="clear"
      className="app-stepper__button"
      onClick={() => setFormData({ ...formData, target_gottesdienst: Math.max(0, formData.target_gottesdienst - 1) })}
    >
      <IonIcon icon={removeOutline} />
    </IonButton>
    <div className="app-stepper__value">
      {formData.target_gottesdienst}
    </div>
    <IonButton
      fill="clear"
      className="app-stepper__button"
      onClick={() => setFormData({ ...formData, target_gottesdienst: formData.target_gottesdienst + 1 })}
    >
      <IonIcon icon={addOutline} />
    </IonButton>
  </div>
  <p className="app-settings-item__subtitle" style={{ textAlign: 'center', margin: '8px 0 0 0' }}>
    Ziel-Punkte für Gottesdienste
  </p>
</IonCardContent>
```

Ersetzen durch:
```tsx
<IonCardContent style={{ padding: '16px' }}>
  <IonItem lines="none" style={{ '--background': 'transparent' }}>
    <div style={{ display: 'flex', alignItems: 'center', gap: '12px', width: '100%' }}>
      <IonButton
        fill="outline"
        size="small"
        disabled={saving || formData.target_gottesdienst <= 0}
        onClick={() => setFormData({ ...formData, target_gottesdienst: Math.max(0, formData.target_gottesdienst - 1) })}
        style={{ '--border-radius': '8px', minWidth: '40px', height: '40px' }}
      >
        <IonIcon icon={removeOutline} />
      </IonButton>
      <IonInput
        type="text"
        inputMode="numeric"
        value={formData.target_gottesdienst.toString()}
        onIonInput={(e) => {
          const val = parseInt(e.detail.value || '0', 10);
          if (!isNaN(val) && val >= 0) {
            setFormData({ ...formData, target_gottesdienst: val });
          }
        }}
        disabled={saving}
        style={{ textAlign: 'center', flex: 1, fontSize: '1.5rem', fontWeight: '700' }}
      />
      <IonButton
        fill="outline"
        size="small"
        disabled={saving}
        onClick={() => setFormData({ ...formData, target_gottesdienst: formData.target_gottesdienst + 1 })}
        style={{ '--border-radius': '8px', minWidth: '40px', height: '40px' }}
      >
        <IonIcon icon={addOutline} />
      </IonButton>
    </div>
  </IonItem>
  <p className="app-settings-item__subtitle" style={{ textAlign: 'center', margin: '8px 0 0 0' }}>
    Ziel-Punkte für Gottesdienste
  </p>
</IonCardContent>
```
